====================================================
Flow Control and Operations
====================================================

Overview
--------

This reading material covers selection statements, iteration statements,
and operations in C++. It is provided as a pre-recorded video on Canvas
and should be watched before starting RWA1.


Learning Objectives
-------------------

By the end of this material, you will be able to:

- Use selection statements (``if``, ``else if``, ``switch``) to control program flow.
- Apply the conditional (ternary) operator for concise expressions.
- Implement iteration statements (``while``, ``do-while``, ``for``) for repetitive tasks.
- Understand operator precedence and associativity.
- Use arithmetic, relational, and logical operators correctly.
- Avoid common pitfalls such as dangling ``else`` and off-by-one errors.
