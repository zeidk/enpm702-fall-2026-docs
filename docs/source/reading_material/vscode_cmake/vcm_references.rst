    .. card::

        **VSCode and CMake**

        Covers configuring Visual Studio Code on Ubuntu to write,
        build, and run C++ programs with CMake. Workspace layout,
        the ``.vscode`` folder, recommended extensions, the Command
        Palette, ``CMakeLists.txt`` essentials, and the hybrid
        per-lecture build setup used throughout the course.
