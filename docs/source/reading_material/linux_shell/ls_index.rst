====================================================
Linux Shell
====================================================

Overview
--------

This reading material introduces the **Linux shell**, the command-line
interface you use to interact with Ubuntu. It is a self-study reading
module. Work through it before the first lecture, since you will use the
shell for every build, run, and ROS 2 workflow in this course. It covers
common shell types, shell configuration files, aliases, and functions.

.. admonition:: Learning Objectives
   :class: learning-objectives

   By the end of this material, you will be able to:

   - Explain what a shell is and identify common shell types (Bash, Zsh, Fish, Csh/Tcsh, Ksh).
   - Determine which shell you are running.
   - Locate and edit shell configuration files such as ``.bashrc`` and ``.zshrc``.
   - Define aliases to shorten frequently used commands.
   - Write simple shell functions that accept and use arguments.
