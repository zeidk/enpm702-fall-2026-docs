    .. card::
        :class-card: sd-border-secondary

        **ENPM702 -- L2: Introduction to C++**

        Covers C++ overview and timeline, the build process (preprocessor,
        compiler, linker), variables (declarations, initializations, naming),
        integral and floating-point types, boolean type, type conversions
        (implicit, promotions, arithmetic), constants (``const``,
        ``constexpr``, macros), type deduction (``auto``), scopes (local,
        global), namespaces, and type aliases.
